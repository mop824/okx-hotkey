# CHANGELOG — TeamMelona OKX Hotkey

OKX 거래소 키보드 단축키 트레이딩 크롬 확장 프로그램. DOM 기반으로 동작하며 API 키 불필요.

---

## [2026-04-09] — 품질 개선 및 기능 추가

### 개선
- `resolveAmount` 공통 헬퍼 추출 — 6개 매수/매도 함수의 중복 로직 통합
- 확인 버튼 폴링 타이밍 수정 — 주문 실행 안정성 향상
- 취소/체이스 액션에 주문 행(row) 폴링 추가 — DOM 준비 전 실행 방지
- 원래 딜레이 값 복원 — 빠른 연속 입력 시 누락 방지
- 중복 단축키 할당 차단 — 동일 키 조합 재등록 불가
- 미사용 코드 및 개발 산출물 제거, `.gitignore` 추가
- `manifest.json` 권한 정리 — 불필요한 퍼미션 제거

### 신규 기능
- **듀얼 사운드 지원** — 청산 액션에 수익/손실 사운드 분리, 매수/매도 액션에 신규진입/물타기 사운드 분리
- **레버리지 연동** — `readLeverage()`로 현재 레버리지 DOM 읽기, 선물 신규 진입 시 레버리지 반영한 수량 계산

## [2026-04-07] — 최초 출시

### 기능
- 시장가 / 지정가 매수·매도 단축키
- **단방향 모드 포지션 인식** — 반대 방향 = 비율 청산, 같은 방향 = 잔고 기반 신규 진입
- 틱 매수/매도 — 1~10호가 오프셋 설정
- 전체 청산, 주문 취소 (마지막 1건 / 전체), 주문 체이스
- 시드캡 — 지정 금액을 100%로 간주한 비율 매매
- 액션별 사운드 알림 (mp3 / wav / ogg)
- 도네이션 모드 — Buy/Sell 버튼을 "Donation"으로 표시
- 드래그 앤 드롭으로 액션 순서 변경
- 현물/선물 자동 감지
