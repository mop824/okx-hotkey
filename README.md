# ⚡ TeamMelona OKX Hotkey

OKX 거래소 키보드 단축키 트레이딩 크롬 확장 프로그램

---

## 주요 기능
- 지정 단축키로 시장가 / 지정가 / 틱 매수·매도 실행 (크롬 기본 단축키는 덮어쓰기 불가)
- 현물 / 선물 , 원웨이모드 / 헤지 모드 지원 (자동 감지)
- TP/SL 자동 설정, 전체 청산, 포지션 반전, 주문 취소, 주문 체이스
- 최대 운용 시드 한도 적용, 액션별 사운드 재생
- 드래그 앤 드롭으로 액션 순서 변경
- **프로필 시스템**: BTC선물 / ETH선물 / 기타 — 코인별로 액션, 시드, TP/SL을 독립 설정
- **거래 페어 자동 감지**: OKX 페이지 접속 시 해당 프로필 자동 적용
- **TP/SL 3가지 모드**: 진입가 %, USDT 금액, 시드 대비 %
- **전역 사운드**: 매매 관련 액션 사운드를 한 번만 설정하면 전체 적용

## 설치
1. 깃헙 다운로드 (초록색 Code 버튼 → Download ZIP → 압축 해제)
2. 크롬 계열 브라우저에서 `chrome://extensions` 접속
3. 우측 상단 "개발자 모드" 활성화
4. "압축해제된 확장 프로그램을 로드합니다" 클릭
5. 압축 해제된 폴더 안의 `dist` 폴더 선택

## 사용법

### 인증
1. [@MelonaManagerTV_bot](https://t.me/MelonaManagerTV_bot) 에 `/start` 보내기 (최초 1회)
2. 크롬 우측 상단 확장 프로그램 아이콘 클릭 → 설정 창 열기
3. "인증" 탭에서 텔레그램 유저네임 입력 후 인증 요청
4. 텔레그램 DM으로 수신된 6자리 인증코드 입력 → 기기 등록 완료 (최대 2대)
- 텔레그램 그룹 멤버만 인증 가능. 그룹 탈퇴 시 자동 회수.

### 단축키 설정
1. "단축키 액션" 탭에서 "액션 추가" 버튼으로 원하는 액션 추가
2. 각 액션에 단축키 설정 (클릭 → 키 입력, Ctrl+Shift+Alt+키, Alt+키 등 조합 가능)
3. 퍼센트(%) 입력: 잔고 또는 포지션의 몇 %를 사용할지 지정
4. 틱 매수/매도: 틱 수(1~1000) 설정으로 호가 오프셋 조절. 예) 틱 매수 5 = 다섯 번째 매수호가에 매수 주문 생성
5. 설정 변경 시 자동 저장됨 — OKX 새로고침 필수 (설정 변경 시마다 동일)

## 액션 목록
- **시장가 매수/매도** — 시장가(Taker) 주문 실행
- **지정가 매수/매도** — 지정가 주문 실행
- **틱 매수** — 최우선 매수호가 + N틱에 지정가 매수 (1틱 = 오더북 최소 호가 단위, OKX BBO Queue 기능)
- **틱 매도** — 최우선 매도호가 - N틱에 지정가 매도
- **전체 청산** — 모든 포지션 시장가 청산
- **페어 청산** — 현재 페이지의 코인 페어 포지션 100% 시장가 청산 (단방향 모드). 예) BTCUSDT Perp 페이지라면 해당 페어만 청산
- **포지션 반전** — OKX 네이티브 Reverse 버튼. 롱↔숏 전환 (Taker)
- **마지막 주문 취소** — 가장 최근 미체결 주문 취소
- **전체 주문 취소** — 모든 미체결 주문 취소
- **주문 체이스** — 미체결 주문을 호가 최상단/최하단으로 지정가 이동 (OKX Chase 기능)
- **롱 시장가 청산** — 롱 포지션의 N% 시장가 청산 (헤지 모드)
- **롱 지정가 청산** — 롱 포지션의 N% 지정가 청산 (헤지 모드)
- **숏 시장가 청산** — 숏 포지션의 N% 시장가 청산 (헤지 모드)
- **숏 지정가 청산** — 숏 포지션의 N% 지정가 청산 (헤지 모드)

## 동작 방식

### 선물 단방향 모드
- 롱 포지션 보유 중 매도 → 포지션의 N% 청산
- 숏 포지션 보유 중 매수 → 포지션의 N% 청산
- 포지션 없거나 같은 방향 → 잔고(또는 최대 운용 시드)의 N%로 신규 진입

### 선물 헤지 모드
- 매수/매도 = 항상 신규 진입
- 롱 청산/숏 청산 액션으로 각 포지션의 % 독립 청산
- 전체 청산 = 롱 + 숏 동시 청산
- 포지션 반전(FLIP), 페어 청산은 헤지 모드에서 사용 불가

### 수량 계산
입력 단위가 코인(ETH, BTC 등)일 때, USDT 금액을 주문 가격으로 나눠 수량을 계산합니다.
- 시장가: 현재 거래 가격 기준
- 지정가: 주문폼 가격 입력란에 적힌 가격 기준
- 틱 매수/매도: 계산된 틱 가격 기준

### 레버리지
OKX 페이지에서 설정한 레버리지 값을 자동으로 읽어와 수량 계산에 반영합니다. 별도 설정 불필요.

## 설정

**최대 운용 시드 (USD)** — 입력한 금액을 100%로 간주해 퍼센티지를 계산. 0이면 Trading 계좌 전체 잔고 기준.
- 예) 시드 10만불, 시드캡 1만불 설정 → 1만불 기준으로 단축키 작동

**TP / SL** — 익절(TP)·손절(SL) 설정. 0이면 비활성화. 시장가/지정가/틱 매수·매도 시 자동으로 TP/SL 가격 입력. 청산/반전/취소 등 포지션 관리 액션에는 적용되지 않음.

3가지 모드 선택 가능:
- **진입가 %** — 진입가 기준 퍼센트. 예) TP 2%, SL 1% → 비트코인 71,000불 롱 진입 시 TP 72,420, SL 70,290 자동 설정
- **USDT 금액** — 절대 금액 기준. 예) TP $200, SL $100 → 해당 금액만큼 이익/손실 발생 시점 가격으로 설정
- **시드 대비 %** — 최대 운용 시드 기준 퍼센트. 예) 시드 10,000불, TP 2% → TP 금액 $200 기준 가격으로 설정

**사운드** — mp3 / wav / ogg 파일 업로드 가능. 액션 실행 성공 시 재생.

**전역 사운드** (매수/매도 관련 액션 전체 적용) — 한 번만 설정하면 모든 매매 액션에 동일하게 적용:
- 신규: 포지션 없을 때 새로 진입
- 물타기: 같은 방향 포지션이 이미 있을 때 추가 진입
- 수익: 반대 포지션을 이익 상태에서 청산
- 손실: 반대 포지션을 손실 상태에서 청산

청산 액션 (2종):
- 수익: 이익 상태에서 청산
- 손실: 손실 상태에서 청산

**개별 사운드** (액션별 독립 설정):
- 주문 취소 / 주문 체이스: 각 액션 카드에서 개별 업로드

기타 액션 (1종): 실행 시 재생

**도네이션 모드** — OKX 페이지의 Buy/Sell 버튼을 "Donation"으로 변경.
