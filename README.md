# TeamMelona OKX Hotkey

OKX 거래소 키보드 단축키 트레이딩 크롬 확장 프로그램

---

## 주요 기능
- 시장가 / 지정가 / 틱 매수·매도 단축키
- 포지션 인식: 반대 방향 매매 시 포지션 비율 청산, 같은 방향 시 잔고 기반 진입
- 틱 매수/매도: 1~10호가 오프셋 설정 가능
- 전체 청산, 포지션 반전 (OKX 네이티브 Reverse)
- 주문 취소 (마지막 1건 / 전체)
- 주문 체이스 (Chase)
- 시드캡 설정 (지정한 한도 금액 내에서 단축키 작동)
- 액션별 사운드 알림 (mp3 / wav / ogg)
- 도네이션 모드 (Buy/Sell 버튼 텍스트 → "Donation")
- 드래그 앤 드롭으로 액션 순서 변경
- 현물/선물 자동 감지, 단방향(one-way) / 헤지(hedge) 모드 지원
- 헤지 모드: 롱/숏 독립 청산 (시장가/지정가)

---

## 설치 방법
1. 이 저장소를 다운로드 (Code → Download ZIP) 또는 `git clone`
2. Chrome에서 `chrome://extensions` 접속
3. 우측 상단 "개발자 모드" 활성화
4. "압축해제된 확장 프로그램을 로드합니다" 클릭
5. 다운로드한 폴더 선택
6. OKX 접속 후 사용

---

## 사용법
1. 크롬 우측 상단 확장 프로그램 아이콘 클릭 → 설정 창 열기
2. "액션 추가" 버튼으로 원하는 액션 추가
3. 각 액션에 단축키 설정 (클릭 → 키 입력, Ctrl / Shift / Alt + 키 조합 가능, 동일한 단축키 조합 중복 할당 불가)
4. 퍼센트(%) 입력: 잔고 또는 포지션의 몇 %를 사용할지 지정
5. 틱 매수/매도: 틱 수(1~10) 설정으로 호가 오프셋 조절
    예) 틱 매수 5 = 다섯번째 매수호가에 매수 주문 생성
6. "저장" 클릭

---

## 액션 목록
1. 시장가 매수/매도 : 현재가로 즉시 매매
2. 지정가 매수/매도 : 현재가 기준 지정가 주문
3. 틱 매수 :         최우선 매수호가 + N틱에 지정가 매수
4. 틱 매도 :         최우선 매도호가 - N틱에 지정가 매도
5. 전체 청산 :       모든 포지션 시장가 청산
6. 페어 청산 :       현재 페어 포지션 100% 시장가 청산 (단방향 모드)
7. 롱 시장가 청산 :  롱 포지션의 N% 시장가 청산 (헤지 모드)
8. 롱 지정가 청산 :  롱 포지션의 N% 지정가 청산 (헤지 모드)
9. 숏 시장가 청산 :  숏 포지션의 N% 시장가 청산 (헤지 모드)
10. 숏 지정가 청산 : 숏 포지션의 N% 지정가 청산 (헤지 모드)
11. 포지션 반전 :    OKX 네이티브 Reverse 버튼으로 롱↔숏 전환
12. 마지막 주문 취소: 가장 최근 미체결 주문 취소
13. 전체 주문 취소 : 모든 미체결 주문 취소
14. 주문 체이스 :    미체결 주문 추적

## 포지션 인식 동작 (선물 단방향 모드)
- 롱 포지션 보유 중 매도 → 포지션의 N% 청산
- 숏 포지션 보유 중 매수 → 포지션의 N% 청산
- 포지션 없거나 같은 방향 → 잔고(또는 시드캡)의 N%로 신규 진입

---

## 헤지 모드 동작 (선물 헤지 모드)
- 매수/매도 = 항상 신규 진입 (포지션 자동 감지 안 함)
- 롱 청산/숏 청산 액션으로 각 포지션 독립 청산
- 전체 청산 = 롱 + 숏 동시 청산
- 포지션 반전(FLIP), 페어 청산은 헤지 모드에서 사용 불가

## 설정
**최대 운용 시드 (USD)**
 값을 입력하면 해당 금액을 100%로 간주해 퍼센트를 계산합니다. 0으로 설정하면 Trading 계좌에 있는 전체 잔고를 기준으로 %를 설정해서 매매합니다.
  예) 시드가 10만불인데 시드캡을 1만불로 설정했으면 1만불을 기준으로 단축키 작동. (1만불의 % 진입)

**레버리지**
OKX 페이지에서 설정한 레버리지 값을 자동으로 읽어와 수량 계산에 반영합니다. 별도 설정 불필요.

**사운드**
각 액션에 mp3 / wav / ogg 파일을 업로드할 수 있으며, 액션 실행 성공 시 재생됩니다.
- 매수/매도 액션: 신규 진입, 물타기, 수익 청산, 손실 청산 각각 다른 사운드 설정 가능
- 청산 액션: 수익, 손실 사운드 분리 설정 가능

**도네이션 모드**
OKX 페이지의 Buy/Sell 버튼을 "Donation"으로 표시합니다.
