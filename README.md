# TeamMelona OKX Hotkey

OKX 거래소 키보드 단축키 트레이딩 크롬 확장 프로그램

---

## 주요 기능
- 지정한 단축키로 트레이딩. 다만 크롬 자체 단축키가 있다면 덮어쓰지 못함 (overide 불가)
- 현물/선물 자동 감지, 단방향(one-way) / 헤지(hedge) 모드 지원
- 시장가 / 지정가 / 틱 매수·매도 단축키
- 포지션 인식: 반대 방향 매매 시 포지션 비율 청산, 같은 방향 시 잔고 기반 진입
- 틱 매수/매도: 1~1000틱 오프셋 설정 가능 (OKX BBO Queue 기능. 1틱 = 해당 코인 오더북의 최소 호가 단위)
- TP/SL 자동 설정: 진입 시 익절/손절 가격 자동 입력 (현물/선물 지원)
- 전체 청산, 포지션 반전 (OKX 네이티브 Reverse 버튼)
- 주문 취소 (마지막 1건 / 전체)
- 주문 체이스 (OKX Chase기능. 미체결 주문을 호가 최상단/최하단으로 지정가 이동)
- 최대 운용 시드 (지정한 한도 금액 내에서 단축키 작동)
- 액션별 사운드 재생 (mp3 / wav / ogg)(뚝배기!)
- 도네이션 모드 (Buy/Sell 버튼 → "Donation")
- 드래그 앤 드롭으로 액션 순서 변경
- 헤지 모드: 롱/숏 독립 청산 (시장가/지정가)

---

## 설치 방법
1. 깃헙 다운로드 (초록색 Code 버튼 → Download ZIP → 압축 해제)
2. 크롬 계열 브라우저에서 "chrome://extensions" 접속
3. 우측 상단 "개발자 모드" 활성화
4. "압축해제된 확장 프로그램을 로드합니다" 클릭
5. 다운로드한 폴더 선택
6. OKX 새로고침 후 사용
7. 단축키 프로그램 설정 변경시 반드시 하단 저장 버튼 클릭 후 OKX 새로고침 필수

---

## 업데이트 방법
1. 새 버전 파일을 기존 폴더에 덮어쓰기
2. "chrome://extensions"에서 확장 프로그램 새로고침 버튼 클릭
3. OKX 탭 새로고침 (F5)

---

## 사용법
1. 크롬 우측 상단 확장 프로그램 아이콘 클릭 → 설정 창 열기
2. "액션 추가" 버튼으로 원하는 액션 추가
3. 각 액션에 단축키 설정 (클릭 → 키 입력, Ctrl+Shift+Alt+키, ALT+키 조합 가능)
4. 퍼센트(%) 입력: 잔고 또는 포지션의 몇 %를 사용할지 지정
5. 틱 매수/매도: 틱 수(1~1000) 설정으로 호가 오프셋 조절. 예) 틱 매수 5 = 다섯번째 매수호가에 매수 주문 생성
6. "저장" 클릭

---

## 액션 목록
- 시장가 매수/매도 : 시장가(Taker) 주문 실행
- 지정가 매수/매도 : 지정가 주문 실행
- 틱 매수 :         최우선 매수호가 + N틱에 지정가 매수
- 틱 매도 :         최우선 매도호가 - N틱에 지정가 매도
- 전체 청산 :       모든 포지션 시장가 청산
- 페어 청산 :       현재 거래 페이지상의 코인 페어 포지션 100% 시장가 청산 (단방향 모드)(예: 현재 띄워져 있는 페이지가 BTCUSDT Perp라면 해당 코인 페어만 시장가 청산)
- 포지션 반전 :     OKX 네이티브 Reverse 버튼. 롱↔숏 전환 (Taker)
- 마지막 주문 취소: 가장 최근 미체결 주문 취소
- 전체 주문 취소 :  모든 미체결 주문 취소
- 주문 체이스 :     미체결 주문 추적
- 롱 시장가 청산 :  롱 포지션의 N% 시장가 청산 (헤지 모드)
- 롱 지정가 청산 :  롱 포지션의 N% 지정가 청산 (헤지 모드)
- 숏 시장가 청산 :  숏 포지션의 N% 시장가 청산 (헤지 모드)
- 숏 지정가 청산 :  숏 포지션의 N% 지정가 청산 (헤지 모드)

---

## 포지션 인식 동작 (선물 단방향 모드)
- 롱 포지션 보유 중 매도 → 포지션의 N% 청산
- 숏 포지션 보유 중 매수 → 포지션의 N% 청산
- 포지션 없거나 같은 방향 → 잔고(또는 최대 운용 시드)의 N%로 "신규 진입"

---

## 헤지 모드 동작 (선물 헤지 모드)
- 매수/매도 = 항상 신규 진입
- 롱 청산/숏 청산 액션으로 각 포지션의 % 독립 청산
- 전체 청산 = 롱 + 숏 동시 청산
- 포지션 반전(FLIP), 페어 청산은 헤지 모드에서 사용 불가

## 설정
- **최대 운용 시드 (USD)**
 값을 입력하면 입력한 금액을 100%로 간주해 퍼센티지를 계산합니다. 0으로 설정하면 Trading 계좌에 있는 전체 잔고를 기준으로 %를 설정해서 매매합니다.
  예) 시드가 10만불인데 시드캡을 1만불로 설정했으면 1만불을 기준으로 단축키 작동. (1만불의 % 진입)

- **레버리지**
OKX 페이지에서 설정한 레버리지 값을 자동으로 읽어와 수량 계산에 반영합니다. 별도 설정 불필요.

- **수량 계산**
입력 단위가 코인(ETH, BTC 등)일 때, USDT 금액을 주문 가격으로 나눠 수량을 계산합니다.
  시장가: 현재 거래 가격 기준
  지정가: 주문폼 가격 입력란에 적힌 가격 기준
  틱 매수/매도: 계산된 틱 가격 기준

- **TP / SL (%)**
익절(TP)과 손절(SL) 퍼센트를 설정합니다. 0이면 비활성화.
시장가/지정가/틱 매수·매도 시 자동으로 TP/SL 가격이 입력됩니다.
  예) TP 2%, SL 1% → 비트코인 71,000불에 롱 진입 시 TP 72,420, SL 70,290 자동 설정
청산/반전/취소 등 기존 포지션 액션에는 적용되지 않습니다.

- **도네이션 모드**
OKX 페이지의 Buy/Sell 버튼을 "Donation"으로 변경합니다.

- **사운드**
각 액션에 mp3 / wav / ogg 파일을 업로드할 수 있으며, 액션 실행 성공 시 재생됩니다.

매수/매도 액션 (4종 사운드):
  신규: 포지션이 없을 때 새로 진입
  물타기: 같은 방향 포지션이 이미 있을 때 추가 진입
  수익: 반대 포지션을 이익 상태에서 청산
  손실: 반대 포지션을 손실 상태에서 청산

청산 액션 (2종 사운드):
  수익: 이익 상태에서 청산
  손실: 손실 상태에서 청산

기타 액션 (1종 사운드):
  실행 시 재생
